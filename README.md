# Social Media Engagement Guide

A research-informed guide explaining how social media engagement works, why it matters, and how it shapes communication, culture, and strategy. Built as a digital learning resource and reflective exploration, the guide integrates academic research, ethical discussion, case analysis, and design-based explanation.

## Features
- Accessible explanation of engagement as design, behavior, and culture
- Organized perspectives: Ethics and Society, Industry and Practice, Emerging Contradictions
- Zotero research library
- Creative micro-interactions and attention-based UI design
- Light and Dark mode toggle with saved preferences
- Responsive layout optimized for mobile and desktop

## Technology
- HTML
- CSS
- Light JavaScript for theme persistence
- Accessible-first design
